# Como Contribuir

Obrigado pelo interesse em contribuir! Veja como você pode ajudar.

## 🐛 Reportando Bugs

1. Verifique se o bug já foi reportado
2. Use o template de issues com:
   - Descrição clara do problema
   - Passos para reproduzir
   - Comportamento esperado vs real
   - Screenshots se possível

## 💡 Sugerindo Features

1. Descreva a feature proposta
2. Explique o caso de uso
3. Seja detalhado nos requisitos

## 🔧 Pull Requests

1. Fork o repositório
2. Crie uma branch (`git checkout -b feature/nova-feature`)
3. Faça commit das mudanças (`git commit -m 'Add nova feature'`)
4. Push para a branch (`git push origin feature/nova-feature`)
5. Abra um Pull Request

## 📝 Padrões de Código

- Use ESLint para linting
- Siga o estilo do código existente
- Adicione comentários quando necessário
- Teste suas mudanças

## ❓ Dúvidas?

Abra uma issue ou entre em contato com a equipe.
