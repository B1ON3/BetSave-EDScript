📋 Resumo do Projeto

Desenvolvimento de um assistente digital para análise de apostas esportivas que transforma dados complexos em recomendações simples e acionáveis.

🎯 Objetivos

  • Analisar partidas esportivas utilizando dados relevantes
  • Gerar previsões de forma compreensível
  • Classificar o nível de risco de cada aposta
  • Explicar os motivos por trás das recomendações
  • Integrar à experiência do usuário de forma simples

🔌 Integração com Esporte da Sorte

Design: O BetSave deve funcionar como uma extensão do site Esporte da Sorte.

  • Botão com ícone no header do site (ao lado do suporte)
  • Ao clicar, abre a interface do BetSave dentro do próprio site
  • Não redireciona - usa iframe/overlay integrado
  • Mesma identidade visual: verde neon (#00ff88), tema escuro
  • Experiência fluida como se fosse parte nativa do Esporte da Sorte

function fetchUrl(targetUrl) {
    return new Promise((resolve, reject) => {
        const protocol = targetUrl.startsWith('https') ? https : http;
        protocol.get(targetUrl, (res) => {
            let data = '';
            res.on('data', (chunk) => { data += chunk; });
            res.on('end', () => { resolve({ statusCode: res.statusCode, data: data }); });
